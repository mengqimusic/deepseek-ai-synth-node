# AI 合成器 Node

## 项目定位

一个可在正交方向上注入能量的单/复音 AI 合成器节点。5 个独立 Voice，各有独立发育史，通过显式能量注入（按键 / patch point）在短/中/长期演化。

## 核心设计文档

- [DESIGN.md](./DESIGN.md) — 系统设计、决策记录、讨论摘要

## 约束

- 默认中文沟通；代码、命令、变量名用英文
- 乐器设计、交互手感、演奏体验的判断以用户为准
- 声音质量是不可妥协的硬约束
- 严格遵循 `可演奏控件六约束`：控制感、差异性、可组合、多样性、确定性、演化性
- 确定性 = 同输入 + 同系统状态 → 同响应。变化来自系统状态的历史积累，不是随机
- 不靠想象毙掉主意；实验结果优先于推理
- git 提交时 Co-Authored-By 写实际使用的模型，当前为: Co-Authored-By: DeepSeek v4 Pro

## 技术方向

- 神经音频直接生成（优先），传统合成 fallback
- 拓扑注入：能量改变神经网络连接路径
- FRAM 存储长期状态
- 存储介质和平台待选型
